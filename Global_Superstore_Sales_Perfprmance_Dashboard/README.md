🌍 Global Superstore Sales Performance Dashboard

📌 Project Overview

The Global Superstore Sales Performance Dashboard is an interactive Power BI solution that provides a comprehensive analysis of worldwide sales, profit, customer segments, shipping performance, and product categories. It enables stakeholders to monitor business performance and make informed decisions using real-time visual insights.

---

🎯 Business Objective

The objective of this project is to:

- Monitor overall sales and profitability.
- Evaluate regional and country-wise business performance.
- Analyze customer segments and purchasing behavior.
- Identify top-performing products and categories.
- Improve business decision-making using interactive dashboards.

---

📊 Dashboard Features

- Executive KPI Cards
- Sales & Profit Analysis
- Regional Performance
- Country-wise Sales
- Category & Sub-Category Analysis
- Customer Segment Analysis
- Shipping Mode Analysis
- Interactive Filters & Slicers

---

📈 Key Performance Indicators (KPIs)

- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin

---

💡 Business Insights

- Identified the most profitable regions and markets.
- Compared category and sub-category performance.
- Evaluated customer segments contributing the highest revenue.
- Analyzed shipping methods to understand operational efficiency.
- Highlighted opportunities to improve profitability.
